This folder should contain local JS libraries:
- tf.min.js (TensorFlow.js)
- teachablemachine-image.min.js (Teachable Machine Image library)

Download them from:
https://cdn.jsdelivr.net/npm/@tensorflow/tfjs@4.20.0/dist/tf.min.js
https://cdn.jsdelivr.net/npm/@teachablemachine/image@0.8.7/dist/teachablemachine-image.min.js

Place both files here before running offline.
